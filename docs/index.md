# Reddit Background Viewer

Welcome to the documentation for the Reddit Background Viewer. This application fetches images from Reddit and displays them as a full-page background.

## Quick Start

The application is a static site. Simply open `404.html` (or `index.html` if available) in a web browser to start viewing background images.

## Features

- **Full-screen background images**: High-quality images sourced from Reddit.
- **Auto-rotation**: Images change every 60 seconds.
- **Subreddit selection**: Specify the subreddit in the URL path.
